berikut merupakan sintaks yang salah
```java
package Array;

public class Array {

  public static void main(String[] args) {
    int [7] arr = {1,2,3,4,5,6 7};

    for(int i=0;i<8;i++){
      System.out.print(arr[i]);
    }
  }
}

```
diberikan sintaks yang salah, tugas kamu adalah membenarkan sintaks yang ada sehingga output yang dihasilkan adalah 1,2,3,4,5,6,7
