berikut merupakan sintaks yang salah
```java
package Array2;

public class Array2 {

  public static void main(String[] args) {
    int arr[] = {1, 2, 3, 4, 6, 6};

    for (int i = 0; i < arr.length; i++) {
      System.out.print(i + 1);
    }
  }
}
```

diberikan sintaks yang salah, tugas kamu adalah membenarkan sintaks yang ada sehingga output yang dihasilkan adalah 1,2,3,4,6,6
